拖拉事件

在网页交中,拖拉事件是非常常见的,如对列表进行排序等操作,如果用操作事件就会非常人性化.那么交互事件分为哪些阶段呢

拖拉 DOM

需要拖拉的目标一共有三个事件:

  1. drag 拖拉过程中,持续触发的事件(相隔几百毫秒)
  2. dragstart 开始拖拉时触发,可通过回调函数中,event的target属性来获取拖拉的 DOM 信息
  3. dragend 结束拖拉时候触发, target是拖拉的 DOM 信息

目标 DOM

  1. dragenter 当拖拉 DOM 进入目标 DOM 时触发,target属性是当前目标 DOM
  2. dragover 进入目标 DOM 上方,持续触发,target节点是当前 DOM
  3. dragleave 离开目标 DOM 时触发,target节点是当前 DOM
  4. drop 被拖拉的 DOM 释放到目标 DOM 时触发,target节点是当前 DOM
